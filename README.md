## 📁 Struktur Folder

```
CV LANDING PAGE PRATAMA/
├── index.html              ← Halaman utama
├── README.md               ← Dokumentasi ini
└── assets/
    ├── css/
    │   └── style.css       ← Stylesheet utama
    ├── js/
    │   └── main.js         ← JavaScript utama
    └── img/
        ├── hero-photo.png         ← Foto profil
        ├── project-taskhub.png    ← Screenshot proyek 1
        ├── project-portfolio.png  ← Screenshot proyek 2
        └── project-library.png    ← Screenshot proyek 3
```

## 🚀 Cara Menjalankan Secara Lokal

Cukup buka file `index.html` di browser:
```
Double click → index.html
```

Atau gunakan ekstensi **Live Server** di VS Code untuk development.

## 🌐 Cara Publish / Deploy

### Opsi 1: GitHub Pages (GRATIS — Direkomendasikan)
1. Buat akun di [github.com](https://github.com)
2. Buat repository baru (contoh: `portfolio`)
3. Upload semua file ke repository tersebut
4. Masuk ke **Settings → Pages**
5. Pilih **Source: Deploy from branch → main → /(root)**
6. Website akan live di: `https://username.github.io/portfolio`

### Opsi 2: Netlify (GRATIS — Termudah)
1. Buat akun di [netlify.com](https://netlify.com)
2. Drag & drop folder `CV LANDING PAGE PRATAMA` ke dashboard Netlify
3. Website langsung live dalam hitungan detik!
4. Dapatkan URL custom seperti: `https://rizkypratama.netlify.app`

### Opsi 3: Vercel (GRATIS)
1. Buat akun di [vercel.com](https://vercel.com)
2. Import dari GitHub atau upload folder
3. Website live otomatis

### Opsi 4: Shared Hosting (Berbayar)
1. Beli hosting + domain di Niagahoster / Domainesia / Hostinger
2. Upload via FileZilla (FTP) ke folder `public_html`
3. Website live di domain Anda sendiri!

## ✏️ Cara Kustomisasi

Edit file `index.html` untuk mengubah:
- **Nama** → Cari `Rizky Pratama` dan ganti
- **Deskripsi** → Ubah teks di section tentang
- **Skills** → Ubah persentase `data-width="90"`
- **Proyek** → Ganti gambar dan deskripsi proyek
- **Kontak** → Ubah email, WhatsApp, dan media sosial
- **Foto** → Ganti file `assets/img/hero-photo.png`

Edit file `assets/css/style.css` untuk mengubah:
- **Warna Utama** → Ubah `--primary: #3b82f6`
- **Font** → Ubah font di `:root`
