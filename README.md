## ✨ Fitur Website

- 🎨 **Desain Premium** — Dark theme dengan glassmorphism dan gradient
- 🖱️ **Custom Cursor** — Cursor interaktif dengan efek hover
- ⚡ **Animasi Smooth** — AOS, typing effect, parallax, skill bars
- 📊 **Counter Animasi** — Statistik angka yang bergerak
- 🔢 **Skill Bars** — Progress bar skill dengan animasi
- 🔍 **Project Filter** — Filter proyek berdasarkan kategori
- 📱 **Responsive Design** — Mobile-friendly di semua perangkat
- 📩 **Form Kontak** — Form dengan validasi dan feedback
- ⬇️ **Download CV** — Tombol download CV

## 🛠️ Teknologi

- HTML5 Semantik
- CSS3 (Custom Properties, Grid, Flexbox, Animasi)
- Vanilla JavaScript (ES6+)
- Font Awesome 6 (Icons)
- Google Fonts (Inter + Outfit)
